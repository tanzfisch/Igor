Welcome to Igor's GitHub Page.

Igor is a non commercial just for the fun of it written game engine. It will probably never be finished and you should not use it in an productive environment. Beyond that please feel free to use it and to have fun with it.

Current Features

  • OpenGL and Windows Support
  • advanced 3D scene graph
  • Octree for fast request
  • general multithreading support
  • simple 2D render support
  • integrated newton game dynamics
  • integrated tinyXML
  • integrated GLee
  • basic widget and dialog system
  • math library vectors, matrices etc.
  • material system
  • full support of OMPF model format
  • loading support 3DS model format
  • loading support OBJ model format
  • comfortable file logging and colorfull console output
  • lot's of example code
  • particles 2D & 3D

Roadmap

  • currently working a lot on voxel based terrains
  • reintegrating sound support (probably using OpenAL) more comming soon ...
