void Igor::iTextureResourceFactory::flush ( iResourceCacheMode  cacheModeLevel = iResourceCacheMode::Free)

works like a garbage collector.

Interrates through all textures and checks how many references every texture has. If reference count goes down to 1 then the texture will be released. If reference count is greater 1 and the texture was not loaded yet and the texture will be loaded.

Parameters
cacheModeLevellevel of cache mode to be released
