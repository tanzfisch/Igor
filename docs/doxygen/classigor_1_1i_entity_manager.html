Detailed Description

entities manager

After experimenting with some ECS (entity component systems) ideas I decided that it is in my case not worth the effort as long as there is not an army of artists and content creators that would work for me. So I stick with the classical approach using inherritance to define entities even if this provokes potencially ugly code within the application layer. I don't care. I want to keep the engine clean.
