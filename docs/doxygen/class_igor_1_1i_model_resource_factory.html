bool Igor::iModelResourceFactory::flush ( iWindow *  window,
iResourceCacheMode  cacheModeLevel = iResourceCacheMode::Free 
)

loads unloads models depending on theirs request status ans reference count

Interrates through all model and checks how many references every model has. If reference count goes down to 1 then the model will be released. If reference count is greater 1 and the model was not loaded yet and the mode will be loaded.

Parameters
windowthe window the resources are bount with
cacheModeLevellevel of cache mode to be released
Returns
true if there was still work to do
